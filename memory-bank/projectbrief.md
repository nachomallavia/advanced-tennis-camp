# Project Brief: Advanced Tennis Camp

## Core Requirements
- Advanced Tennis Camp website built with Astro.
- Multi-language support (French, English).
- Modern UI with animations and interactive components.
- Vercel deployment.

## Goals
- Provide information about tennis camps, programs, and prices.
- Allow users to contact the camp.
- Showcase the camp through galleries and videos.

## Scope
- Frontend: Astro, Tailwind CSS, React.
- Animations: Anime.js, Framer Motion (planned).
- UI Library: Starwind UI (custom).
