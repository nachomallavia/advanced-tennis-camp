# Tech Context: Advanced Tennis Camp

## Technologies Used
- **Astro**: Main framework.
- **React**: Interactivity.
- **Tailwind CSS**: Styling (v4).
- **TypeScript**: Type safety.
- **Anime.js**: Animations.
- **Vercel**: Deployment.
- **Lucide**: Icons (Astro and React versions).

## Development Setup
- Node.js environment.
- Package Manager: npm (implied by `package-lock.json`).
- Shell: bash.

## Technical Constraints
- Must maintain performance while adding animations.
- Ensure cross-browser compatibility for direction-aware effects.
- Multi-language routing considerations.
